## Tech Stack

**Bankend:** Java, SpringBoot \
**Database:** MySql 



## Run Locally

Clone the project

```bash
  git clone https://github.com/ags4436/account-management-system
```

Go to the project directory

```bash
    Import Project as existing maven Project
```

maven Update 

```bash
  mvn clean install -U
```

Start the server

```bash
  Run as SpringBoot Application
```
